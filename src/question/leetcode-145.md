# 145.Binary Tree Postorder Traversal 
Given the root of a binary tree, return the postorder traversal of its nodes' values.

### Example 1:
![pre1](../pre1.jpg)
``` 
Input: root = [1,null,2,3]
Output: [3,2,1]
```
### Example 2:
``` 
Input: root = []
Output: []
```
### Example 3:
``` 
Input: root = [1]
Output: [1]
```
### Example 4:
![pre3](../pre3.jpg)
``` 
Input: root = [1,2]
Output: [2,1]
```
### Example 5:
![pre2](../pre2.jpg)
``` 
Input: root = [1,null,2]
Output: [2,1]
```
