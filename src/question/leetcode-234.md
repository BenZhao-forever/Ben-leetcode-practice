# 234.Palindrome Linked List
Given the head of a singly linked list, return true if it is a palindrome.

### Example 1:
![pal1linked](../pal1linked-list.jpg)
``` 
Input: head = [1,2,2,1]
Output: true
```
### Example 2:
![pal2linked](../pal2linked-list.jpg)
``` 
Input: head = [1,2]
Output: false
```
